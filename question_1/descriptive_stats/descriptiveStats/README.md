# descriptiveStats

The goal of descriptiveStats is to provide a range of functions to
enable basic numerical analysis (including finding the mean, median,
mode and interquartile range)

Package installation can be done directly by calling:
‘devtools::install(“question_1/descriptive_stats”)
library(descriptiveStats)’

# example

data \<- c(1, 2, 2, 3, 4, 5, 5, 5, 6, 10) calc_mean(data)
